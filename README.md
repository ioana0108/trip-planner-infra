# Trip Planner – Microservices Project

This project is part of the IDP course and represents a simple Trip Planner application built using a microservices architecture.

## Services

- **Auth Service** – used for user registration and login
- **Trip Service** – used for creating trips and viewing existing trips

## Technologies
- ASP.NET Core Web API
- Docker & Docker Compose

## How to run

```bash
docker compose up --build
